Various Computational Photography Projects
Using various vision based algorithms to modify existing photographs.
Tuesday, September 8, 2015 / Updated November 30, 2015

Overview
All of the projects are listed below. They link to my Github with the source code, and some explanation given the appropriate project. This was my favorite set of projects to work on. Part of me wants to understand how things work (iPhone panoramas, face morphing, etc) and these set of projects help me reverse engineer and understand what is really going on behind the scenes.
